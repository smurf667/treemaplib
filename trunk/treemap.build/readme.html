Introduction

You'll find the project hosted at https://treemaplib.googlecode.com/.
To build with Maven, you need to check out:
  • treemap.build (this project)
  • treemap.core
  • treemap.examples
  • treemap.swing
  • treemap.swing.examples
  • treemap.swt
  • treemap.swt.examples
  • treemap.eclipse
To build, issue mvn clean install here.

Examples

After you've built you can run the file viewer example:
  • Swing: mvn initialize -f run-fileviewer-swing.xml
  • SWT: mvn initialize -f run-fileviewer-swt.xml
